## Installation ⚒️

1. Clone the repository:
   
   ```bash
   git clone https://github.com/abhinavkumar2369/Python-Snake-Game.git
   ```
   
3. Navigate to the project directory:
   
   ```bash
   cd Python-Snake-Game
   ```


<!------------------------------------------------->



## Usage 🤖

1. Run the script:

   ```bash
   python Source_Code.py
   ```
